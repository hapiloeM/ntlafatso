# Ntlafatso Consulting Website (Static)

This is a simple, fast static website you can upload anywhere (cPanel, Netlify, Vercel, GitHub Pages).

## Files
- index.html
- about.html
- services.html
- brochure.html
- contact.html
- css/styles.css
- js/main.js
- assets/favicon.svg

## How to run locally
Open `index.html` in your browser.

## How to upload
Upload the entire folder contents to your web hosting root (often `public_html/`).

## Images
Images are loaded from Unsplash Source (placeholder images). You can replace them with your own by:
1) putting images in `/assets/`
2) updating the `<img src="...">` paths in the HTML.
